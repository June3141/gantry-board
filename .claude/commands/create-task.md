指定された機能を分析し、実装タスクに分割してください。

## 手順

1. 指定された機能の要件を docs/PRD.md から確認する
2. 機能を実装可能な小さなタスクに分割する
3. 各タスクについて以下を定義する:
   - タイトル
   - 説明（受け入れ条件を含む）
   - 優先度 (low / medium / high / urgent)
   - 依存関係（他のタスクに依存する場合）
   - 推定テストケース

## 出力形式

各タスクを以下の形式で出力:

```
### タスク: [タイトル]
- **説明**: ...
- **優先度**: ...
- **依存**: ...
- **テストケース**:
  - [ ] ...
  - [ ] ...
```
